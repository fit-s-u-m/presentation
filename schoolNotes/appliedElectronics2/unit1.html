# Feed back amplifiers --- Feed back amplifier : -
*Is one in which the fraction of the output is given back to the input* --- **If the output is given back to the input increased the input it's called**

positive feed back

direct feed back

regenerative feed back

If the output is 180 degree out of phase with the input or if the output reduce the input in feed back it's called

Negetive feed back

inverse feed back

degenerative feed back

**Normal amplifiers takes an input and mutiplies the input and make an output** - $v_G = \frac{v_o}{v_i}$
This is called **open gain**
%%{init: {'theme': 'dark', 'themeVariables': { 'darkMode': true }}}%% flowchart TD A[input] --> B[Amplifier] B --> C[output] C --> D[beta factor] D --> A
But with feed-back loop since the output is the output is feedback to the input multiplied with $\beta ^*$ and feed back to the input again so
$v_i = v_i + \beta v_o$ --- when this gets amplified so the new output is
$v_o = A(v_i + \beta v_o)$
$v_o - v_o(A \beta) = A v_i $
$v_o(1-A\beta)= A v_i$
$v_o = \frac {A v_i} {1- A\beta}$ --- so the amplification is $A = \frac {v_o}{v_i} $
$\bold{A}= \frac{A}{1-A\beta}$ --- here if
$A\beta = 1$ ***this means mathematically we have infinite amplification meaning we get an output without input***

and for the negative feed back loop the output is subtracted from the input $v_i = v_i - \beta v_0$
therefore $A= \frac{A}{1+A\beta}$

# gain stability if we differentiate the $\bold{A'}= \frac{A}{1-A\beta}$ we get $\frac{dA'}{A'} = \frac{1}{A\beta} \frac{dA}{A}$
since $A\beta >1$ we decrease the change so negative feed back give us better stability
